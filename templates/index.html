Что уже учтено в заготовке

Уже есть Flask-маршруты, Bootstrap-интерфейс и шаблоны. Это позволяет дальше поэтапно внедрять БД, API и загрузку файлов без переделки главной страницы.

SQLAlchemy-модели, Flask-WTF формы, вход и регистрацию, а затем REST API на Flask-RESTful для управления тарифами и подписками.
